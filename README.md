## Installation
```sh
$ git clone https://github.com/TacioAntonio/angular-cineflix
$ cd angular-cineflix
$ npm i
```

## Running locally
```sh
$ npm start
```

## Running on docker
```sh
$ docker-compose up
```

## Running unit tests
```sh
$ npm run test
```
